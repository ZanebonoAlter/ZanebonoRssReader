# Topics界面设计调整方案

## 核心调整点

### 1. 数据展示逻辑重构
- 事件/人物/关键词**直接关联具体的articles**（原始日报内容）
- AI总结仅作为引用这些标签的概括
- 用户优先看到的是**真实的日报内容时间线**

### 2. 拓扑图连线交互优化
- **默认状态**: 只展示节点，**完全不显示任何连线**
- **选中题材后**: **动态绘制**与该题材相关的连线
- **视觉效果优化**: 贝塞尔曲线、渐变色、动画效果、发光效果

### 3. 拓扑图下方新增"日报时间线"区域
- 垂直时间轴形式（左侧时间，右侧内容）
- 日报内容卡片：标题、摘要、发布时间、来源feed
- 按时间倒序排列
- AI分析入口（默认收起）

### 4. 右侧详情栏精简优化
- 只展示带有对应关键tag的articles（严格去重）
- 相关主题改为"关键词云"形式
- 点击关键词仅高亮拓扑图节点

### 5. AI分析功能的重新定位
- **默认只展示日报内容**（不展示AI分析）
- **AI分析作为增强功能**，默认收起
- 提供"AI深度分析"按钮
- 分析对象：当前选中的关键元素
- 展示位置：时间线区域下方独立面板（可展开/收起）

## 实施优先级

### P0 - 核心结构调整
1. 拓扑图连线交互改为"选中后展示"
2. 拓扑图下方新增"日报时间线"区域
3. 数据流调整：事件/人物/关键词直接关联articles

### P1 - 右侧栏精简
1. 右侧栏去重逻辑
2. 关键词云形式展示
3. 点击只高亮拓扑节点

### P2 - AI分析增强
1. AI分析按钮
2. AI分析面板
3. 增量存储分析结果

## 需要确认的问题

1. **日报时间线的展示数量**: 默认展示多少条？支持分页/无限滚动吗？
2. **AI分析的触发时机**: 用户点击按钮后立即调用AI？还是后台预生成？
3. **拓扑图连线的动画效果**: 具体想要什么样的动画（生长动画/淡入淡出/其他）？
4. **关键词云的展示**: 展示多少个关键词？如何确定字号大小？
5. **日报时间线和右侧栏的去重**: 基于article ID去重，还有其他维度吗？
