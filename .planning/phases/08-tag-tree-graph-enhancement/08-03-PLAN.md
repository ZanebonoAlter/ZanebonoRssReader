       - 如果 `timeRange` 非空，计算起始时间：
         ```go
         var since time.Time
         switch timeRange {
         case "7d": since = time.Now().AddDate(0, 0, -7)
         case "30d": since = time.Now().AddDate(0, 0, -30)
         default: since = time.Time{} // no filter
         }
         ```
       - 查询在时间范围内有文章关联的标签 ID 集合：
         ```go
         var activeTagIDs []uint
         database.DB.Model(&models.AISummaryTopic{}).
             Joins("JOIN articles ON articles.summary_id = ai_summary_topics.summary_id").
             Where("articles.published_at >= ?", since).
             Pluck("DISTINCT ai_summary_topics.topic_tag_id", &activeTagIDs)
         ```
         注意：AISummaryTopic 关联的是 summary_id，需要检查实际的文章关联方式。可能需要通过 summaries → articles 链接。检查 AISummary 是否有 article_id 或 feed_item_id 外键。
       - 构建 `isActiveMap map[uint]bool`。
       - 在构建 TagHierarchyNode 时设置 `IsActive`。

    4. 修改 handler 读取 `time_range` query param 并传递：
       ```go
       timeRange := c.Query("time_range")
       nodes, err := GetTagHierarchy(category, feedID, categoryID, timeRange)
       ```

    5. 更新 GetTagHierarchy 的所有调用方传入空字符串 `""` 作为默认 timeRange。
