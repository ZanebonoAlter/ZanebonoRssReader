## Milestone Summary

**Key Accomplishments:**

1. **pgvector 向量搜索基础设施** — 标签 embedding 从 JSON 迁移到 pgvector vector 列，SQL 级余弦距离搜索
2. **三级标签匹配 + 自动收敛** — exact → alias → embedding，新文章入库时实时合并高相似度标签
3. **关注标签 + 首页推送** — 心形图标关注、侧边栏分组、相关度排序文章流
4. **标签合并交互界面** — 全量扫描预览、自定义名称合并、批量操作确认
5. **抽象标签提取 + 层级树** — LLM 从中间地带(0.78-0.97)提取共同概念，递归层级展示
6. **标签树增强与图谱优化** — Description 生成、时间筛选、3D 发光可视化、节点手动归类

**Key Decisions:**

- 复用 airouter provider 框架（CapabilityEmbedding），不自建 embedding 模块
- 自动合并标签（非聚类展示），从根本上简化标签空间
- 新文章入库时实时触发收敛，非定时批量
- Middle band (0.78-0.97) 跳过 AI 判定，降级为创建新标签后由抽象标签机制处理
- Phase 3/4/5 因设计方向变更跳过

**Issues Resolved:**

- PostgreSQL JSON 数组别名查询函数错误
- TagMergePreview 渲染问题（GlobalSettings 内嵌）
- 抽象标签图谱节点未标注 IsAbstract
- 自定义时间范围格式解析 fallback

**Technical Debt:**

- 标签 quality_score 与收敛流程的联动需后续完善
- 日志门面未覆盖所有调度器（仅覆盖 auto_refresh/auto_tag_merge/content_completion）
